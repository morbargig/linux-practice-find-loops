## Goal
Learn to process text data using `cut`, `sort`, and `uniq` in pipelines.

## Tasks
1. Extract columns from structured data
2. Sort data alphabetically and numerically
3. Remove duplicates with `uniq`
4. Combine tools in pipelines

## Deliverable
Create `pipeline_results.txt` with:
- Output of each pipeline command
- Explanation of what each tool does
- Combined pipeline examples

## Hints
- Use `cut -d:` to split on colon delimiter
- Use `sort` to order data
- Use `uniq` to remove consecutive duplicates
- Combine with pipes: `command1 | command2 | command3`
