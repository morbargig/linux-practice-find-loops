# Exercise 06 — Safe Sudo Configuration (visudo)

## Goal
Learn the correct way to edit sudo permissions.

## Tasks
1. Open sudoers safely using `visudo`
2. Find where user permissions are defined
3. Explain what could go wrong if you edit sudoers with a normal text editor.

## Deliverable
Create `visudo-report.md`:
- 3–5 bullets explaining:
  - what `visudo` does
  - why syntax matters
  - what happens if sudoers breaks

## Hints
- Do NOT change anything unless your instructor asked you to.
- This is a reading + understanding task.
