# Exercise 01 — Identify Yourself (whoami, id)

## Goal
Learn how to check the current user and understand user/group IDs.

## Tasks
1. Run:
   - `whoami`
   - `id`
2. Explain (in your own words) what each of these means:
   - UID
   - GID
   - groups

## Deliverable
Create a file named `answers.txt` and write:
- The output of `whoami`
- The output of `id`
- 2–3 lines explaining UID/GID/groups

## Hints
- UID is the numeric ID of the user.
- A user can belong to multiple groups.

## Docs
- `id` command: https://man7.org/linux/man-pages/man1/id.1.html
- `whoami` command: https://man7.org/linux/man-pages/man1/whoami.1.html
