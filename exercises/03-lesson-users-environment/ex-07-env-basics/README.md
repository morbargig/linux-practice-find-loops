# Exercise 07 — Environment Variables (env, printenv)

## Goal
Understand what environment variables are and why they matter.

## Tasks
1. Print these variables:
   - `USER`
   - `HOME`
   - `SHELL`
   - `PATH`
2. Compare the output between your user and `techstudent`.

## Deliverable
Create `env-compare.md`:
- outputs for both users
- 2–3 lines: what changed and why?

## Hints
- Use `printenv VAR_NAME`
- Use `su - techstudent` to compare "login environment".

- printenv: https://man7.org/linux/man-pages/man1/printenv.1.html
